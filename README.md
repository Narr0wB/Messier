# Messier

Chess Engine written in C++.

## Getting startedÂ 

First, clone the repository

```sh
$ git clone https://github.com/Narr0wB/Messier
$ cd Messier
```

To build the project, make sure you have the following tools installed:

- make

Then

```sh
$ make
$ ./build/Messier
```

## Usage

The Engine implements the standard version of the UCI (Universal Chess Interface) protocol as defined [here](https://official-stockfish.github.io/docs/stockfish-wiki/UCI-&-Commands.html).
